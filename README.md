  ---
  打牌记账小程序

  一款专为麻将、扑克、斗地主等牌局设计的微信小程序，支持多人房间实时记账与个人快速记账两种模式。

  功能特性

  - 房间模式：创建牌局房间，生成6位邀请码，支持扫码或分享链接加入
  - 实时同步：基于云数据库 Watch API，所有玩家得分实时推送（延迟 <1s）
  - 双结算模式：支持按点数（含底分倍率）或直接金额两种方式结算
  - 排行榜：房间内玩家余额实时排名
  - 好友系统：记录与每位好友的对局历史、胜率及净盈亏
  - 统计页面：总对局数、胜率、总盈亏、场均盈亏
  - 个人记账：无需创建房间的快速单人记账模式
  - 分数校验：提交前实时验证所有玩家分数之和为零

  快速开始

  1. 克隆本仓库
  2. 使用微信开发者工具打开项目
  3. 在 project.config.json 中替换为你自己的 AppID
  4. 开通微信云开发并创建对应数据库集合（参考 DATABASE.md）
  5. 部署云函数
  6. 编译运行

  环境要求

  - 微信开发者工具
  - 基础库版本 >= 2.32.3
  - 已开通微信云开发
